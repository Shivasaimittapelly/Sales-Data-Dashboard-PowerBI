# Sales Performance Dashboard - Power BI

## Project Overview

This project analyzes sales performance using Power BI. The dashboard provides insights into regional sales, category-wise revenue distribution, product profitability, and key business KPIs.

## Objectives

* Visualize sales performance
* Track business KPIs
* Identify top-performing products
* Compare regional sales
* Support data-driven decision making

## Tools Used

* Power BI Desktop
* Microsoft Excel
* DAX Measures

## KPIs

* Total Sales
* Total Profit
* Total Orders
* Total Quantity
* Profit Margin

## Dashboard Features

### Regional Sales Analysis

Displays sales performance across different regions.

### Category Sales Distribution

Shows contribution of each product category to total sales.

### Product Profit Analysis

Highlights the most profitable products.

### Interactive Filtering

Users can filter data by region and category using slicers.

## Business Insights

* West region generated the highest sales.
* Electronics contributed the largest share of revenue.
* Laptop was the highest profit-generating product.
* Total sales reached 437K with 82K profit.
